About the Application

MyTodo is a powerful task management application designed to help you stay organized and focused. With MyTodo, you can create tasks, mark them as complete, update their details, and track your progress with intuitive visualizations.

This application was built using Flask, a lightweight WSGI web application framework in Python, along with SQLAlchemy for database management. The frontend was developed using Bootstrap to ensure a responsive and user-friendly interface across all devices.

Key Features

  • Create, update, and delete tasks
  • Mark tasks as complete with visual indicators
  • Track your progress with a visual dashboard
  • Secure user authentication and personalized task lists
  • Responsive design for all devices
